Приложение - мессенджер, созданное в процессе изучения python, flask, sqlalchemy, docker и т.д.

Основной функционал:

* Добавление пользователей в контакты по никнейму
* Настройки - изменение логина, пароля, email, статуса
* Общение между двумя пользователями
* Групповые беседы с несколькими пользователями (beta)

Функции, работающие без перезагрузки страницы:

* отправка сообщений
* Получение новых сообщений
* Подгрузка старых сообщений при скроле вверх
* Обновление информации о количестве непрочитанных сообщений
* Обновление информации о всех диалогах
* Получение информации о пользователе при клике на никнейм
